C/C++ Beginner Quiz

2 marks each question

1. What is the correct syntax for declaring a variable in C?

a) int myVariable;
b) var myVariable = 0;
c) let myVariable;

2. What is the difference between int and float data types in C?

a) int is used to store whole numbers, while float is used to store decimal numbers.
b) int is more precise than float.
c) int is faster than float.

3. What is the difference between a function and a macro in C?

a) A function can have parameters, while a macro cannot
b) A function is a block of code that can be reused, while a macro is a single line of code that is expanded at compile time
c) A function can return a value, while a macro cannot.

4. What is the difference between #include and #define directives in C?

a) #include is used to include header files, while #define is used to define macros.
b) #include directives are processed at compile time, while #define directives are processed by the preprocessor.
c) #include directives can be used to include multiple files, while #define directives can only be used to define a single macro.

5. What is the difference between main() and int main(void) in C?

a) main() is a function that must be present in every C program, while int main(void) is a function that can be used to return a value.
b) main() is the entry point of all C programs, while int main(void) is a function that is called by the operating system when the program is executed
c) main() can have parameters, while int main(void) cannot.

6. What is the difference between a class and a struct in C++?

a) A class is a blueprint for creating objects, while a struct is a data structure
b) A class can have member functions, while a struct cannot.
c) A class can have protected and private members, while a struct cannot

7. What is the difference between a constructor and a destructor in C++?

a) A constructor is used to initialize the properties of an object, while a destructor is used to clean up the resources used by an object.
b) A constructor is called when an object is created, while a destructor is called when an object is destroyed.
c) Both constructors and destructors can have parameters.

8. What is inheritance in C++?

a) Inheritance is a mechanism that allows us to create new classes that inherit the properties and methods of existing classes.
b) Inheritance is a way to reuse code and create more complex objects
c) Inheritance is a fundamental concept in object-oriented programming.

9. What is polymorphism in C++?

a) Polymorphism is the ability of an object to take on many different forms.
b) Polymorphism is the ability of a method to be overridden by subclasses.
c) Polymorphism is the ability of a variable to hold objects of different types.

10. What is the difference between a reference and a pointer in C++?

a) A reference must be initialized when it is declared, while a pointer does not need to be initialized.
b) A reference cannot be reassigned, while a pointer can be reassigned.
c) Both
