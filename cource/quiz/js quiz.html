JAVASCRIPT Beginner Quiz

2 marks each question

1. Which of the following is not a valid JavaScript data type?

a) Number
b) Symbol
c) String

2. Which of the following is a valid JavaScript variable name?

a) my-variable
b) my_variable
c) Both

3. Which of the following is a valid JavaScript function declaration?

a) function myFunction() { }
b) let myFunction = function() { }
c) Both

4. Which of the following is a valid JavaScript conditional statement?

a) if (condition) { statement }
b) if (condition) { statement } else { statement }
c) Both

5. Which of the following is a valid JavaScript loop statement?

a) while (condition) { statement }
b) do { statement } while (condition);
c) Both

6. Which of the following is a valid JavaScript array method?

a) push()
b) pop()
c) Both

7. Which of the following is the correct syntax for creating a new object in JavaScript?

a) const obj = {};
b) var obj = {};
c) Both

8. What is the difference between == and === in JavaScript?

a) == performs loose equality checking, while === performs strict equality checking
b) == checks if the values of two operands are equal, while === checks if the values and types of two operands are equal
c) Both

9. What is the difference between null and undefined in JavaScript?

a) null represents the intentional absence of a value, while undefined represents a value that has not yet been assigned.
b) null is a global variable, while undefined is a local variable.
c) null is a primitive data type, while undefined is a reference data type.

10. What is the purpose of the this keyword in JavaScript?

a) The this keyword refers to the current object.
b) The this keyword can be used to access the properties and methods of the current object.
c) The this keyword can be used to pass the current object as an argument to a function
